# Investment Manager

## Vision

Investment Manager is a modern full-stack web application built with Rust that enables users to securely manage, monitor, and analyze their investment portfolios.

The application is designed for individual investors who want a simple, secure, and efficient platform for tracking investments such as stocks, ETFs, mutual funds, bonds, and cryptocurrencies.

The primary goals are:

- Secure user authentication
- Portfolio management
- Investment tracking
- Portfolio analytics
- Responsive web interface
- Clean architecture
- High performance

## Objectives

- Build a production-quality Rust application.
- Demonstrate modern backend engineering practices.
- Provide an intuitive investment dashboard.
- Create a portfolio-quality software project.

## Target Users

- Individual investors
- Students learning investing
- Rust developers studying web development
