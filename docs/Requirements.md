# Requirements

## Functional Requirements

### User Management

- Register account
- Login
- Logout
- Update profile

### Portfolio

- Create portfolio
- Add investments
- Remove investments
- Edit investments

### Assets

- Create assets
- View assets
- Update assets
- Delete assets

### Dashboard

- Display total portfolio value
- Display gain/loss
- Display number of investments
- Display asset allocation

### Authentication

- Password hashing
- JWT authentication
- Secure cookies

- Responsive UI
- Secure authentication
- Modular architecture
- PostgreSQL database
- Docker support
- Fast API responses
- Error handling
- Logging
